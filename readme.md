## 1. ツール概要 (Overview)
VRC Expression Editorは、VRChatアバターの **「1フレーム表情アニメーション」を直感的に作成・編集することに特化したUnity拡張エディタ** です。
Unity標準のアニメーションウィンドウでの煩わしい作業を排除し、スライダーを動かすだけでリアルタイムに表情を作り込むことができます。

### ✨ 表情作りを加速させる強力な便利機能
ただシェイプキーを動かすだけでなく、アバター改変の面倒な作業を直感的なUIとワンクリックで解決する機能を搭載しています。

* **🔍 圧倒的に探しやすい「絞り込み＆ソート」**
数百個のシェイプキー地獄から解放！ 複数キーワードで探せる「詳細フィルター」や、よく使うキーの「★ピン留め」、編集中のキーが上にくる「動いている順ソート」で目当てのキーを瞬時に見つけ出します。
* **👀 リアルタイム・プレビュー**
専用ウィンドウで実際の変形を見ながらスライダー調整。シーンの照明も自動で反映されます。
* **🤝 別メッシュ連動 ＆ 左右対称（ミラー）編集**
「顔と体」など別メッシュの同名シェイプキーを自動で同時操作。L・Rの左右対称編集にも対応。
* **💥 表情干渉テスト**
「リップシンク」や「ベースの顔」など、他のアニメーションを重ねて再生し、顔が破綻しないか安全にテストできます。
* **🛡️ ワンクリック「0埋め」処理**
Write Defaults OFF環境などで必須となる「未登録キーの0埋め」や「参照アニメからのコピー」をボタン一つで一括実行。
* **🔄 スナップショット比較**
作っている途中の表情を一時保存し、「変更前・変更後」をワンタッチで見比べる・復元することができます。
* **📷 メニュー用アイコンのワンタッチ撮影**
作った表情を、EXメニューで使いやすい「背景透過PNGアイコン」としてその場で撮影・保存できます。
