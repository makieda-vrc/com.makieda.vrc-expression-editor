## 1. ツール概要 (Overview)
VRC Expression Editorは、VRChat向けアバターの **「1フレーム表情アニメーション」を作成・編集するためのUnity拡張エディタ** です。
Unity標準のアニメーションウィンドウでの複雑な階層操作を介さず、専用 of UIからスライダーを用いてリアルタイムに表情のアニメーションクリップを編集できます。

### ■ アバター改変における利便性を高める設計
シェイプキーの直接操作に加え、表情作成における一連の改変・設定作業を効率化するための以下の機能を備えています。

* **🖼️ 視覚的な「表情一覧（サムネイルグリッド）」**
  FXコントローラー等のレイヤーに含まれる表情アニメーションを自動で抽出し、サムネイル付きでグリッド表示します。アニメーションがどのレイヤーのどのステートに設定されているかも一目で把握でき、目的のアニメーションを視覚的に選択してすぐに編集を開始できます。
* **🔍 オブジェクトとシェイプキーの絞り込み・ソート**
  複数キーワードに対応した「詳細フィルター」、使用頻度の高いキーを固定する「★お気に入り（ピン留め）」、値が変動しているキーを優先的に上位表示する「動いている順ソート」等により、目的の項目を迅速に探索できます。
* **👀 リアルタイム・プレビューと照明同期**
  独立したプレビューウィンドウを介して表情の変形を確認できます。シーン上の照明環境（Directional Lightや環境光）が自動で同期・反映されます。
* **🤝 別メッシュ連動 ＆ 左右対称（ミラー）編集**
  「Face」と「Body」など、異なるメッシュに配置された同名のシェイプキーを同時に操作できます。また、L・Rの接尾辞を検知した左右対称の同時編集に対応しています。
* **💥 表情干渉・ブレンドテスト**
  「リップシンク」や「ベース表情」などの他アニメーションを下層に重ねて再生し、作成中の表情と干渉した際に表示が破綻しないかを検証できます。
* **🛠️ 表情データの一括処理（0埋め・項目コピー）**
  Write DefaultsがOFFの環境等で推奨される「未登録キーの0埋め（初期化）」や、既存のアニメーションクリップから項目構成のみをコピーする処理を一括で実行できます。
* **🔄 一時スナップショット比較**
  作成中の表情データをメモリ上に一時保存し、現在の編集状態とワンタッチで比較、または保存時点の状態への復元が可能です。
* **📷 透過PNGアイコンの撮影**
  作成した表情を、Action Menu（EXメニュー）用スプライト等として再利用可能な「背景透過PNG画像」としてプレビュー画面から直接撮影・保存できます。

## 3. 使い方 (Usage)

### ■ 基本的な導入手順
1. Unityのメニューバーから `Tools` > `VRC Expression Editor` を選択してウィンドウを開きます。
2. `表情エディタ` および `表情プレビュー` ウィンドウが起動します。
3. エディタ上部の「アバター」スロットに、編集対象となるHierarchy上のアバターオブジェクトをドラッグ＆ドロップします。

### ■ 表情一覧（サムネイルグリッド）の活用
1. 表情エディタの「アニメ」選択スロットの右隣にある **格子状の「グリッドアイコン 🖼️」** をクリックし、`表情一覧` ウィンドウを起動します。
2. アバターに登録されているアニメーションコントローラー（FXなど）から、対象レイヤー内の表情アニメーションが自動的に抽出され、サムネイル付きで一覧表示されます。
3. サムネイルの下部には、そのアニメーションがFXコントローラー内の「どのレイヤー」の「どのステート」に配置されているかが表示されます（表示のON/OFFは右下の「S」ボタンで切り替え可能）。
4. 表示サイズは、右下のスライダー、または右端の数値入力フィールドから自由に拡大・縮小が可能です。
5. 撮影画像がまだ生成されていない場合や、表情を修正した後に再撮影したい場合は、サムネイル上で **右クリック** をして「この表情を再撮影」を選択することで、現在のライティング環境が反映された透過PNG画像が再生成・保存されます。

### ■ 詳細フィルター（OR検索・複数キーワード絞り込み）の活用
1. 検索フィルタテキストボックスの右隣にある **歯車（設定）アイコン ⚙️** をクリックし、画面左側の `詳細フィルター` パネルを開閉します。
2. 「＋ 条件を追加」ボタンをクリックするとキーワード入力行が追加されます。
3. 追加した入力行にキーワード（例：「eye」「smile」「まばたき」等）を入力します。登録されたキーワードのいずれかに部分一致するシェイプキーが、メイン画面上に同時に抽出されます（複数キーワードによるOR検索）。
4. 各キーワードの左側にあるチェックボックスを外すことで、その条件のみを一時的に無効化できます。不要になった入力行は、右端の「✕」ボタンで削除できます。
5. 詳細フィルターパネルとメイン操作画面の境界線をドラッグすることで、パネルの横幅をお好みのサイズに調整できます。
