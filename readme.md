## 1. ツール概要 (Overview)
VRC Expression Editorは、VRChat向けアバターの **「表情アニメーション」を作成・編集するためのUnity拡張エディタ** です。
専用 of UIからスライダーを用いてリアルタイムに表情のアニメーションクリップを編集できます。
表情タイムラインを併用することで、滑らかな表情変化のアニメーション作成もサポートします。

## 3. 使い方 (Usage)

### ■ 基本的な導入手順
1. Unityのメニューバーから `Tools` > `VRC Expression Editor` を選択してウィンドウを開きます。
2. `表情エディタ` および `表情プレビュー` ウィンドウが起動します。
3. 編集対象となるHierarchy上のアバターオブジェクトを選択します。
4. 対象メッシュに表情のシェイプキーが含まれるメッシュを選択します。(例、Body)
5. 対象レイヤーでFXのhandレイヤーを選択します。(例、Left Hand,Right Hand)
6. 絞り込まれたアニメから編集したいアニメを選択します。

### ■ 表情一覧（サムネイルグリッド）の活用
1. 表情エディタの「アニメ」選択スロットの右隣にある **格子状の「グリッドアイコン」** をクリックし、`表情一覧` ウィンドウを起動します。
2. 対象レイヤー内の表情アニメーションが自動的に抽出され、サムネイル付きで一覧表示されます。
3. サムネイルの下部には、そのアニメーションがFXコントローラー内の「どのレイヤー」の「どのステート」に配置されているかが表示されます（表示のON/OFFは右下の「S」ボタンで切り替え可能）。
4. 表示サイズは、右下のスライダー、または右端の数値入力フィールドから自由に拡大・縮小が可能です。
5. 撮影画像がまだ生成されていない場合や、表情を修正した後に再撮影したい場合は、サムネイル上で **右クリック** をして「この表情を再撮影」を選択することで、現在のライティング環境が反映された透過PNG画像が再生成・保存されます。

### ■ 表情タイムライン（Animation Timeline）の活用
1. Unityのメニューバーから `Tools` > `VRC Expression Editor` > `Animation Timeline` を選択し、タイムラインウィンドウを起動します。
2. アニメーションが選択されると、タイムライン上に登録されているキーフレームが「白い縦線」および「◇」で描画されます。
3. **キーフレームの移動**:
   * 動かしたいダイヤ（◆）を左クリックで掴み、左右にドラッグして任意のフレーム（時間）でマウスを離します。
   * 「キー吸着」ボタンが有効（緑色）になっている場合、近くのキーフレーム位置にぴったりと吸着（スナップ）します。
4. **複数キーの一括移動**:
   * キーが配置されていない余白部分からマウスをドラッグすると、青い「範囲選択ボックス」が表示されます。
   * ボックスで囲んだ複数のキーをまとめて選択し、選択枠の内側をドラッグすることで、タイミングを保ったまままとめて平行移動できます。
5. **対象絞り込み**:
   * 表情エディタウィンドウのシェイプキー名をクリックすることでタイムラインの情報を選択キーだけに絞り込みます。解除はもう一度クリックです。また、Shiftキーを押しながら複数追加・除外することも可能です。

### ■ 詳細フィルター（OR検索・複数キーワード絞り込み）の活用
1. 検索フィルタテキストボックスの右隣にある **歯車（設定）アイコン ⚙️** をクリックし、画面左側の `詳細フィルター` パネルを開閉します。
2. 「＋ 条件を追加」ボタンをクリックするとキーワード入力行が追加されます。
3. 追加した入力行にキーワード（例：「eye_」「smile」「まばたき」等）を入力します。登録されたキーワードのいずれかに部分一致するシェイプキーが、メイン画面上に同時に抽出されます（複数キーワードによるOR検索）。
4. 各キーワードの左側にあるチェックボックスを外すことで、その条件のみを一時的に無効化できます。不要になった入力行は、右端の「✕」ボタンで削除できます。
5. 詳細フィルターパネルとメイン操作画面の境界線をドラッグすることで、パネルの横幅をお好みのサイズに調整できます。
